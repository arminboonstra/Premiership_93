This application gives an overview of the English Premiership table for the 1992-1993 season.

The season consisted of 22 teams. Manchester United were the champions with 92 points. Swindom finished bottom with 30 points.

The application allows the user to see how the number of points per team developed through the season. 
The user van select which teams he or she wants selected in the graph. 
The user is also able to select the graph up to a specific game week.
